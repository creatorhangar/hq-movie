# 🎬 SKILL: Auditoria de Capa para TikTok/Reels

## Objetivo
Validar e otimizar capas de vídeo para máximo engajamento em TikTok, Reels e Stories.

---

## 📊 Pesquisa Adobe/TikTok (300 thumbnails analisados)

### Cores que Performam Melhor
| Cor | Views | Engagement |
|-----|-------|------------|
| Marrom | #1 | Alto |
| Verde | Alto | #2 comentários/shares |
| Rosa | Alto | Baixo engagement |
| Laranja | Médio | #1 engagement (12.3%) |
| Roxo | Médio | #2 engagement (12.2%) |

### Texto na Thumbnail
- **5-10 palavras** = sweet spot máximo engagement
- **Posição centro-inferior** = 13% engagement (melhor)
- **Posição topo** = 11.2% engagement (pior)
- **+15 palavras** = drástica redução de engagement

### Emoções que Engajam
1. Surpresa
2. Raiva  
3. Felicidade
4. Humor

## ✅ Checklist de Auditoria

### 1. SAFE ZONES TikTok (CRÍTICO)
- [ ] Username overlay (esquerda inferior): 60px margem
- [ ] Botões laterais (direita): 80px margem
- [ ] Descrição (inferior): 150px margem
- [ ] Barra de progresso: 10px margem inferior
- [ ] **Área segura**: Centro-superior 70% da tela

### 2. LEGIBILIDADE
- [ ] Título com **mínimo 48px** em 1080x1920
- [ ] Contraste texto/fundo > 4.5:1 (WCAG AA)
- [ ] Stroke/shadow no texto para legibilidade
- [ ] Fonte bold para títulos
- [ ] Máximo 2 fontes diferentes

### 3. HIERARQUIA VISUAL
- [ ] Título = elemento dominante (60%+ área visual)
- [ ] Subtítulo 50-60% do tamanho do título
- [ ] Metadados (autor, vol) discretos (< 12px)

### 4. DURAÇÃO/FRAME
- [ ] Capa = primeiro frame do vídeo OU
- [ ] Capa = frame customizado (não animado)
- [ ] Para TikTok: usar como "thumbnail estático" em grid
- [ ] Duração recomendada capa: 0.5-1s no início

### 5. CALL-TO-ACTION VISUAL
- [ ] Elemento de curiosidade/mistério
- [ ] Emoção clara no visual
- [ ] Texto provocativo (pergunta, afirmação bold)

## 🔴 Problemas Encontrados (ANTES)

### Dashboard
1. ❌ **Sem botão de idioma** - usuário não consegue trocar idioma na tela inicial
2. ❌ Traduções faltando em algumas áreas

### Modo Capa
1. ❌ **Sem indicador de safe zones TikTok** - texto pode ficar sob UI
2. ❌ Preview muito pequeno (37% zoom)
3. ❌ Sem preview mobile (como vai aparecer no feed)
4. ❌ Templates não otimizados para TikTok safe zones
5. ❌ Sem contador de palavras do título
6. ❌ Sem preview de contraste/legibilidade
7. ❌ Duração da capa não configurável separadamente

### Exportação
1. ❌ Capa não tem opção de ser frame inicial do vídeo
2. ❌ Sem export específico para thumbnail TikTok (1080x1920 estático)

## 📐 Especificações Técnicas

### Dimensões TikTok
- **Vertical**: 1080 × 1920px (9:16)
- **Safe Area Centro**: 810 × 1440px (75% interno)
- **Margem lateral**: 135px cada lado
- **Margem inferior**: 240px (UI do TikTok)
- **Margem superior**: 120px

### Frame Rate
- TikTok suporta: 24, 25, 30, 50, 60 fps
- Recomendado: 30fps

### Duração Mínima
- TikTok: 1 segundo
- Reels: 3 segundos
- Stories: 1-15 segundos
