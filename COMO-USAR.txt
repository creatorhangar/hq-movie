🇧🇷 PORTUGUÊS

═══ OPÇÃO 1: USO SIMPLES (SEM INSTALAÇÃO) ═══

1. Descompacte o arquivo ZIP em qualquer pasta do seu computador
2. Abra o arquivo "index.html" com Google Chrome ou Microsoft Edge
3. Pronto! O aplicativo funcionará diretamente no navegador

IMPORTANTE:
• Use Chrome ou Edge para melhor compatibilidade
• Funciona em Windows, Mac e Linux
• Seus projetos são salvos automaticamente no navegador
• Use "Exportar Projeto" (arquivo .hq) para fazer backup


═══ OPÇÃO 2: SERVIDOR LOCAL (MELHOR QUALIDADE DE VÍDEO) ═══

Para exportação de vídeo com máxima qualidade, use um servidor local:

1. Instale Python 3 (se ainda não tiver):
   • Windows/Mac: Baixe em python.org
   • Linux: Já vem instalado na maioria das distribuições

2. Abra o terminal/prompt de comando na pasta do HQ Movie

3. Execute o comando:
   python3 serve-nocache.py

4. Abra o navegador e acesse:
   http://localhost:8082

5. Para parar o servidor, pressione Ctrl+C no terminal


═══ RECURSOS PRINCIPAIS ═══

✓ Crie motion comics com áudio e narração
✓ 34 layouts de página prontos
✓ Balões de fala, pensamento, grito e sussurro
✓ Ken Burns (zoom/pan em imagens)
✓ Exportação de vídeo (MP4/WebM)
✓ Exportação PDF, PNG/JPG e HTML
✓ Multilíngue (Português e Inglês)
✓ Funciona 100% offline


═══ DICAS ═══

• Pressione ESPAÇO para play/pause no preview
• Use Ctrl+Z para desfazer
• Clique com botão direito para opções rápidas
• Faça backup regular dos seus projetos (.hq)


═══ REQUISITOS MÍNIMOS ═══

• Navegador: Chrome 90+ ou Edge 90+
• RAM: 4GB (8GB recomendado para vídeos)
• Espaço: 200MB
• Sistema: Windows 7+, macOS 10.13+, Linux (qualquer)
